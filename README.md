# Keilyndas - E-commerce de Servicios de Manicura

Keilyndas es una plataforma web completa orientada a la gestión de citas y servicios de manicura. El proyecto está diseñado para ofrecer una experiencia de usuario fluida, desde la exploración de estilos hasta la reserva de citas y la consulta de políticas de atención.

## Estructura de la Aplicación

- **Inicio:** Landing page de bienvenida.
- **Servicios:** Catálogo detallado de tratamientos.
- **Normas:** Políticas de higiene y bienestar.
- **Cuidados:** Guía post-servicio para el cliente.
- **Galería:** Muestra visual del portafolio.
- **Preguntas:** Preguntas frecuentes (FAQ en ingles) para reducir dudas frecuentes.
- **Contacto:** Canales de atención directa (Email, IG, WhatsApp).
- **Registro/Login:** Sistema de autenticación seguro para gestionar el perfil del usuario, historial de citas y preferencias personalizadas.
- **ARMA TU ESTILO:** Portal de gestión para clientes registrados.
- **Términos, Condiciones y Políticas de Servicio**, garantizando transparencia legal y seguridad para el cliente.

## Tecnologías

- React
- Next.js
- CSS-in-JS (Estilos integrados)

## Instrucciones de Instalación

1. Clonar el repositorio:
   ```bash
   git clone [https://github.com/LuisinhoGozzo/Keilyndas_proyecto_Gozzo.git](https://github.com/LuisinhoGozzo/Keilyndas_proyecto_Gozzo.git)
   cd Keilyndas_proyecto_Gozzo

2. Instalar dependencias:
Bash
npm install

3. Iniciar el proyecto:
Bash
npm run dev

4. Acceso:
Abre http://localhost:3000 en tu navegador web.

Nota de seguridad: Para el despliegue configuré las variables de entorno (.env.local) requeridas en la plataforma Vercel, cumpliendo con las buenas prácticas de seguridad para no exponer credenciales en el repositorio público.
